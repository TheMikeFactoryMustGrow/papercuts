# New repo checklist (fleet / personal)

When starting or cloning a project you will work on with agents:

1. [ ] `papercut enable` (or: “set up papercuts in this repo”)  
2. [ ] Confirm `papercut status` → `project_enabled=true`  
3. [ ] Commit `AGENTS.md` snippet + `.agent-papercuts/` (open.md, history.jsonl, README)  
4. [ ] Optional: if the repo still has a Claude-only brief, plan a **platform-agnostic** pass (`AGENTS.md` primary, thin `CLAUDE.md` shim) next time you work there — do not block enablement on that cleanup  

Do **not** put ambient papercuts logging in home/global AGENTS.
